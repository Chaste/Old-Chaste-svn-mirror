Welcome to Chaste.

This README file is for the Chaste standalone cardiac simulation executable.

Select a suitable directory where to place the files and cd into it.
Unpack the downloaded compressed executable.
For an XX bits machine and revision number YYYY, type

tar xvfz StandaloneChaste_rYYYY_XXbit.tgz

There are 5 files:
1) StandaloneChaste_rYYYY_XXbit/README
2) StandaloneChaste_rYYYY_XXbit/Chaste	(the executable file)
3) StandaloneChaste_rYYYY_XXbit/ChasteDefaults.xml (contains some default values for a simulation)
4) StandaloneChaste_rYYYY_XXbit/ChasteParameters.xml (where to specify all the parameters for a simulation e.g., mesh, cell models, output directory etc.)
5) StandaloneChaste_rYYYY_XXbit/ChasteParameters.xsd (xml schema. Please, do not edit this file)

In order to run a simulation edit the ChasteParameters.xml file according to your needs and, 
from the installation directory, type

./Chaste ChasteParameters.xml

Please note that the output directory specified in the ChasteParameters.xml file is 
relative to a directory defined by the environmental variable CHASTE_TEST_OUTPUT. 
In order to set the CHASTE_TEST_OUTPUT to a desired location:

export CHASTE_TEST_OUTPUT=Desired_Directory_Path

If CHASTE_TEST_OUTPUT is not set, results will be found relative to a 'testoutput' 
folder in the current directory.
